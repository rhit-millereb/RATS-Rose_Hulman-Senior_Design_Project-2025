TAK Server

This folder contains the files necessary to create a TAK Server on a raspberry pi running piOS.

It contains two objects:

1) "TAK" Folder
	- Contains files necessary to run the installation process
	- Contains the .deb file to install official TAK dependencies
	- Contains bash scripts to help with the installation process
	- Contains instructions on how to use the contents in the folder

2) "takserver.img" file
	- An img of the hard drive of the raspberry pi with takserver installed
	- Can be used as a backup img of the existing pi
	- Can be written onto a 16GB SD card and used to duplicate the server
